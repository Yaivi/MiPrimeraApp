# MiPrimeraApp
Este Codelab lo considero de los más útiles, ya que es el primero que empieza a explicar como usar el IDE de Android Studio, enseñando como crear un proyecto, la forma de ejecutar una app emulando un dispositivo, y las cosas que diferencian a Kotlin como un lenguaje centrado en dispositivos móviles con respecto a otros lenguajes de programación. Enseña como funciona el Preview para ver en tiempo real como va cambiando cada uno de los elementos que creemos y sirve para afianzar los conceptos que se han explicado anteriormente en los Codelabs más introductorios.
